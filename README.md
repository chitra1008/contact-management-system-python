# Contact Management System (Python)

A simple **command-line Contact Management System** built using Python.  
This program allows users to manage contacts efficiently with options to add, view, search, update, and delete contact information.

---

## Features

- Add new contacts with name, phone, email, and address  
- View all saved contacts in a numbered list  
- Search contacts by name or phone number  
- Update existing contact details  
- Delete contacts  
- Menu-driven interface for easy navigation  

---

## Technologies Used

- Python 3  
- Lists and Dictionaries  
- Functions and Loops  
- Exception Handling
